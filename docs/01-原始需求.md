# 后端开发面试任务 - 原始需求

> 来源：飞书面试任务文档
> 截止时间：5月17日（周日）下午 14:00 前提交

---

## 1. 核心目标

基于 **Django/FastAPI** + **MySQL/PostgreSQL** + **Redis** 框架，完成一个后端登录接口 + 业务接口的编写，并且撰写详细的接口文档。

**注意**：项目在本地可运行即可。面试官的测试方式是：
- `git clone` 到本地仓库
- 根据 md 文档进行本地运行
- 用 Postman 调用接口进行测试

---

## 2. 业务接口

- **接口入参**：无

**接口实现**：
基于（抓包）美国地址生成器 https://www.meiguodizhi.com/ 网站中的生成地址接口：

```
POST https://www.meiguodizhi.com/api/v1/dz
```

在本项目中调用该接口，并把成功返回信息中的 `address` 字段进行返回。

**接口返回格式**：
```json
{"code": 200, "data": {"address": {}}, "msg": "success"}
```

---

## 3. 登录接口与鉴权

1. 基于数据库设计用户表（用户 id、账号 account、密码 pwd、name 即可）
2. 实现登录接口，入参账号和密码，成功后返回 token，token 存储在 redis 中
3. 实现接口鉴权，基于 token 身份鉴权实现，接口鉴权是对所有请求的接口进行判断，并且设计白名单功能

---

## 4. 编写接口文档

编写详细的接口文档，对接口的调用方式进行说明，形成 `.md` 文件。

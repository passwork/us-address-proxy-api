## 二、外部 API 探测结论

### 2.1 真实请求方式

```http
POST https://www.meiguodizhi.com/api/v1/dz
Content-Type: application/json
Referer: https://www.meiguodizhi.com/

{
  "path": "/",
  "method": "address"
}
```

> **发现过程**：通过分析首页 JS 源码 `py.3979d900d7da9f788296.js`，定位到 `firstFetchAddressData()` 函数的构造逻辑为 `JSON.stringify({path: window.location.pathname, method: "address"})`。

### 2.2 成功响应结构

```json
{
  "address": {
    "Address": "4427 Kincheloe Road",
    "City": "Portland",
    "State": "OR",
    "State_Full": "Oregon",
    "Zip_Code": "97232",
    "Telephone": "503-527-5486",
    "Full_Name": "Benai",
    "Gender": "Female",
    "Credit_Card_Number": "4916977754095978",
    ...
  },
  "status": "ok"
}
```

### 2.3 关键实现要点

1. **Referer 必须带**：该接口校验 `Referer: https://www.meiguodizhi.com/`，否则可能被拦截
2. **请求体固定**：后端内部转发时 Body 写死为 `{"path": "/", "method": "address"}`，无需前端传参
3. **提取字段**：从外部响应中取出 `response["address"]`，包装进统一响应格式
4. **健壮性**：外部服务偶发超时或 5xx，建议给 `httpx` 设置 `timeout=10s`，并做 `try/except` 捕获

## 三、面试官关注的技术亮点（加分项）

1. **密码安全**：使用 `bcrypt` 而非 `md5/sha256`
2. **Token 设计**：建议用**随机高熵字符串**（32 字节以上）存 Redis，而非 JWT —— 面试场景下更易解释过期与主动失效；若用 JWT，需说明 Refresh Token 机制
3. **异步全链路**：FastAPI + async SQLAlchemy + httpx，展示对 Python 异步 IO 的理解
4. **配置管理**：使用 `pydantic-settings` + `.env` 文件，硬编码是减分项
5. **异常处理**：全局异常捕获中间件，确保任何错误都返回统一 JSON 格式，不会抛 500 HTML
6. **外部 API 健壮性**：超时设置（10s）、连接失败降级、Mock 数据兜底
7. **白名单可扩展**：即使常量实现，也封装成函数/配置，说明"未来可接入 Admin 动态配置"

## 四、实施计划

建议总耗时 **13 小时**，分 **3 天**完成。

| 阶段 | 任务 | 预估耗时 | 产出 |
|------|------|----------|------|
| **Day 1 上午** | 外部 API 探测、技术选型确认、项目脚手架搭建（FastAPI + SQLAlchemy async + Alembic + Redis 连接） | 3h | 可运行的空项目 |
| **Day 1 下午** | 用户 Model/迁移、注册初始化数据（seed）、登录接口（bcrypt + Token + Redis 存储） | 3h | 登录接口可测 |
| **Day 2 上午** | 全局鉴权中间件（Dependency）、白名单机制、Token 过期策略 | 3h | 鉴权体系完整 |
| **Day 2 下午** | 业务接口（httpx 异步调用外部 API、异常处理、Mock 降级）、统一响应封装 | 2h | 全部接口可测 |
| **Day 3 上午** | Postman 测试全链路（登录→拿 Token→调业务→Token 失效→白名单绕过）、Bug 修复 | 2h | 稳定版本 |
| **Day 3 中午** | 编写 `README.md`（环境准备、运行步骤）+ `docs/api.md`（接口文档）、Git 提交、邮件发送 | 2h | 最终交付物 |
